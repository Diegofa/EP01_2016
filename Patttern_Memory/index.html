PATTERN MEMORY - GAME 

                
Instrucciones

                Este juego consiste en visualizar el patron indicado,
                luego de unos segundos este se ocultará y el jugador deberá seleccionarlos tal y como los vio. 

                
A tener en cuenta
     
                Hay un tiempo determinado para visualizar el patròn, y un tiempo para seleccionarlo tal como se vio. 
                A medida que se avance será mas dificil el juego. 
                BUENA SUERTE!!
